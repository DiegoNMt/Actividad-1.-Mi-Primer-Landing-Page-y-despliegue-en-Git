Sobre Mí

Conoce más sobre mi trayectoria escolar

Soy estudiante de desarrollo web y tecnología, enfocado en seguir aprendiendo y fortaleciendo mis habilidades para crear soluciones digitales funcionales. Tengo experiencia trabajando con tecnologías frontend como HTML, CSS y JavaScript, mientras continúo ampliando mis conocimientos en el desarrollo de aplicaciones web.

También estoy desarrollando habilidades en backend utilizando Python y bases de datos como MySQL, PostgreSQL y MongoDB. Me interesa crear aplicaciones web modernas, responsivas y bien estructuradas que ofrezcan una buena experiencia de usuario.

Durante mi formación utilizo herramientas como Git y GitHub para control de versiones, Docker para entornos de desarrollo y plataformas en la nube como AWS y Azure. Además, complemento mis proyectos con diseño y prototipado en Figma y animaciones básicas en CSS, siempre buscando mejorar como desarrollador mientras continúo con mi carrera.
